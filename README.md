# Install
1. Copy your favorite .xccolortheme file to
`~/Library/Developer/Xcode/UserData/FontAndColorThemes`

2. Restart Xcode

3. Open Xcode `preferences` menu, and `Fonts & Colors` tab

4. Select the color theme you added from left pane
